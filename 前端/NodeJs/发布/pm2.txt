详情
	https://github.com/Unitech/PM2/
安装
	hnpm i pm2 -g
查看版本
	pm2 -v
查看应用列表
	pm2 list
查看单个应用详情
	pm2 show 0 //id=0,not pid
	pm2 show app-koa2sms //name="app-koa2sms"
停止应用
	pm2 stop app-koa2sms //name="app-koa2sms"
停止并删除应用
	pm2 delete all
	pm2 delete 0 //Delete app with id 0,not pid 0
	pm2 delete app-koa2sms //name="app-koa2sms"
启动
	直接启动文件
		pm2 start bin/index.js --node-args="--harmony"
	启动配置文件
		pm2 start process.yml
		配置文件
			apps:
			  - script   : bin/index.js
			    name     : app-koa2sms
			    watch    : true
			    # instances: 2
			    node_args: --harmony
			    exec_mode: fork
			    #error_file: /log/err.log
			    #out_file: /log/out.log
			    env :
			      PORT: 3000
			      NODE_ENV: dev
	启动指定启用
		pm2 start app-koa2sms //name="app-koa2sms"
		pm2 start 0 //id=0,not pid
