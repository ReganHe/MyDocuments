-v 查看版本
compile 编译
test 测试
package 打包

clean 清空target
install 安装Jar包到仓库中，并编译，测试

创建目录的两种方式：
1.archetype:generate 按照提示进行选择
2.archetype:generate 
	-DgroupId=组织名,公司网址的反写+项目名
	-DartifactId=项目名-模块名
	-Dversion=版本号
	-Dpackage=代码所存在的包名

坐标：groupId + artifactId + version
构件：依赖、插件、项目构件的输出
仓库：本地仓库和远程仓库
	mvn提供了一个默认的全球中央仓库：https://repo.maven.apache.org/maven2
镜像仓库：相当于远程仓库的一个拷贝
